# Cornerstone Catering Services

A full-stack Nigerian food ordering platform built for a real catering 
business serving customers in Chicago and Minnesota. Live and actively 
used by real customers.

## Live Demo
[cornerstonecatering.vercel.app](https://cornerstonecatering.vercel.app)

## Tech Stack
Next.js · TypeScript · MongoDB · Stripe · Tailwind CSS · Cloudinary · Nodemailer · JWT

## Features
- Meal browsing by category with variant/size pricing
- Stripe checkout with payment processing and webhooks
- Live order tracking with timeline UI
- Automated delivery status emails at every order stage
- Full admin dashboard — manage meals, categories, users and orders
- Role-based authentication (admin vs customer)
- ZIP code delivery zone validation
- Cloudinary image uploads for meal management

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
